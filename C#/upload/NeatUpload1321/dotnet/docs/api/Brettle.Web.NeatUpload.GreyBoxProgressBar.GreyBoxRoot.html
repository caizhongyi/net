GreyBoxProgressBar.GreyBoxRoot Property

URL of the directory containing the GreyBox support files.

public string GreyBoxRoot {get; set;}

Remarks

The URL should not include a the trailing "/". You may use an absolute or relative URL that refers to a page in the same web application. If the URL starts with "~", the "~" will be replaced with the web application root as returned by ApplicationPath. By default, "~/greybox" will be used.
