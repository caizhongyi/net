UploadStateStoreProvider.IsStale Method 

Returns true if the specified UploadState is considered stale.

protected bool IsStale(
   UploadState uploadState
);

Parameters

uploadState
The UploadState to check for staleness

Return Value

true if uploadState is stale.

Remarks

The UploadState is considered stale if it has not been updated in the number of seconds indicated by the stateStaleAfterSeconds attribute of the <neatUpload> element or it has been forced stale.
