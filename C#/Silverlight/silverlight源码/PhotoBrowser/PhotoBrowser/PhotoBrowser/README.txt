To search for photos using this application, you'll need to get a 
Flickr API key from http://www.flickr.com/services/api/keys/apply/ 
and then add it to Flickr.cs

The earth image was taken from 
http://planetpixelemporium.com/planets.html and is copyright
James Hastings-Trew

You can use the F2-F6 keys to switch between views of the application.
