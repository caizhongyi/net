MSMQ Order Processing sample

Demonstrates how to configure destinations with the MSMQ Adapter.

Before running this sample please create 3 private queues in the Microsoft Management Console:

orderqueuein - Flex Producer objects will place orders in this queue (illustrates using messages serialized with BinaryMessageFormatter)

orderqueuenotifications - Flex Consumer objects will receive order notification from this queue (illustrates using messages serialized with XmlMessageFormatter). Aditionally illustrates how to filter messages with message selectors.

backoffice - Flex Consumer objects will receive order notification from this queue. In the sample the same Flex application subscribes to the queue without using message filtering so messages will be broadcasted to all Consumers subscribed to this destination.

 

The OrderProcessing WinForm application is responsible to read messages from the orderqueuein queue, set the Id property for each OrderVO and send them to the orderqueuein and backoffice queues.
