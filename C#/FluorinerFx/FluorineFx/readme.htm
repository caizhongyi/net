Installation

The FluorineFx Installation Wizard creates the following Windows registry keys in a typical install situation:
HKEY_LOCAL_MACHINE\SOFTWARE\FluorineFx (installation path)
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\.NETFramework\AssemblyFolders\FluorineFx 1.1
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\.NETFramework\AssemblyFolders\FluorineFx 2.0
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\.NETFramework\AssemblyFolders\FluorineFx 3.5
These registry keys are required only for the Visual Studio wizards/templates.

Installing the FluorineFx Service

This option will install the FluorineFx Service as a Windows service. The FluorineFx Service provides RPC, Messaging and Data Services over Rtmp.

The FluorineFx Service can be controlled using the Windows Service Control Manager (Control Panel\Administrative Tools) or by using NET commands from the command line. By default the FluorineFx Service startup type is Manual (will not start automatically when Windows starts).

You can also run the FluorineFx Service from the command line with the following command:
[FluorineFx Installdir]\Server\FluorineFx.exe -Console

QuickStart Samples

A comprehensive set of Quickstart samples has been created for this release in the C# language.
These are located in the \Program Files\FluorineFx\Samples directory. Please refer to the samples readme.htm file in the \Samples directory for instructions on building and configuring the samples.
